# Cæcilie Balling

Official website for Cæcilie Balling.

## Publishing

The site is a static website published from the `main` branch through GitHub Pages. The primary address is `www.caecilieballing.com`.

For the first migration stage, the domain remains registered at Wix and its DNS records are managed there. Only the website A/CNAME records should be changed; Google Workspace mail records must be preserved. Cloudflare can be introduced after the domain is transferred away from Wix.

## Working locally

Open `index.html` directly in a browser, or serve the folder with any local static web server.
